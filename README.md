# Hotel Booking Analytics Dashboard

## Project Overview

This project analyzes hotel booking data using Power BI and SQL to provide insights into booking performance, revenue, customer behavior, transaction status, and customer satisfaction.

---

## Objectives

- Analyze booking trends
- Monitor hotel revenue
- Evaluate customer ratings
- Track transaction performance
- Identify top-performing hotels
- Support business decision-making

---

## Dataset

The project consists of five related tables:

- Hotels
- Bookings
- Users
- Transactions
- Ratings

---

## Tools Used

- Power BI
- SQL
- DAX
- Excel

---

## Dashboard Pages

### 1. Hotel Booking Overview

Features:

- Total Hotels
- Total Bookings
- Total Revenue
- Average Stay Days
- Revenue by City
- Top 10 Hotels by Revenue
- Bookings by Room Type

### 2. Customer & Transaction Insights

Features:

- Total Transactions
- Completed Transactions
- Pending Transactions
- Transaction Success Rate
- Average Rating
- Positive Rating %
- Revenue by Age Group
- Rating Distribution

---

## Key Insights

- Mumbai generated the highest revenue.
- Luxury rooms received the most bookings.
- Average stay duration was 5 days.
- Transaction success rate reached 89%.
- Positive customer ratings accounted for 65%.
